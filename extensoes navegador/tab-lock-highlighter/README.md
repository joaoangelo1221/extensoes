# Tab Switcher + Locker + Highlighter (MV3)

Extensão com três funções:
1) **Atalhos para alternar abas**: Ctrl+< (esquerda) e Ctrl+> (direita) mapeados como `Ctrl+Shift+Comma` e `Ctrl+Shift+Period`.
2) **Bloqueio por senha** de uma aba específica ou de todas as abas, com **desfoque** e sobreposição para desbloqueio.
3) **Realce de texto** selecionado com cor configurável e botão para limpar os destaques.

## Como instalar
1. Abra `chrome://extensions/` no Google Chrome.
2. Ative o **Modo do desenvolvedor** (canto superior direito).
3. Clique em **Carregar sem compactação** e selecione a pasta `tab-lock-highlighter` (após extrair o ZIP).

## Atalhos de teclado
- Sugeridos: **Ctrl+Shift+,** (Ctrl+<) para aba à esquerda e **Ctrl+Shift+.** (Ctrl+>) para aba à direita.
- Se quiser mudar, abra `chrome://extensions/shortcuts` e ajuste os atalhos dos comandos da extensão.

## Bloqueio com senha
- Defina a senha no popup da extensão.
- **Bloquear esta aba** apenas a aba atual.
- **Bloquear todas** aplica a todas as abas da janela atual.
- **Observação**: isto é um bloqueio *visual* (ofuscação) — não substitui controles de segurança reais.
